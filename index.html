Smart Stress Monitoring System

2025

Developed an IoT-based Smart Stress Monitoring System using physiological sensors such as EMG and GSR to capture real-time stress data, which is logged to Google Sheets for storage and analysis. The system focuses on continuous stress monitoring and the detection of abnormal stress patterns. Machine learning models, including Isolation Forest and Autoencoders, were used for anomaly detection, while XGBoost was applied for stress-related disorder prediction based on the collected sensor data.
